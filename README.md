### 🌐 Overview Profissional

Bacharel em Sistemas de Informação focado no desenvolvimento de arquiteturas eficientes, governança de sistemas e garantia de qualidade (QA). Minha atuação é centrada em mitigar riscos técnicos, otimizar fluxos de processos e assegurar a integridade do ecossistema de software de ponta a ponta. 

Com uma trajetória consolidada por **9 anos de liderança e disciplina operacional na Marinha do Brasil**, trago uma sólida capacidade de gerenciamento de crises, tomada de decisão sob alta pressão e coordenação técnica. Atualmente, aplico essa visão analítica avançada na automação de processos, auditoria de dados e engenharia de software voltada para soluções de negócios reais.
